# CrossHelp

Chrome Extension combining multiple utilities:

## Features

- **Math Copying** - Copy math formulas from web pages
- **Markdown Viewer** - View .md files directly in browser
- **New Tab Page** - Custom NTP with u2re.space embedded

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
